# ASCII-DAG
[![CI/CD](https://github.com/samsmithnz/ASCII-DAG/actions/workflows/workflow.yml/badge.svg)](https://github.com/samsmithnz/ASCII-DAG/actions/workflows/workflow.yml)

A POC to see if we can create ASCII DAG graphs. 

Some examples: 
```
A--B

A--B------D--E
   \     /
    --C--

   B--C
  /
 /
A
 \
  \  
   D--E

A--B--D--E--F
    \   /  
     \ /
      C
```
